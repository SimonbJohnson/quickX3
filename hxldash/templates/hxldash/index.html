HXLDash (Alpha)

Dashboards optimised for humanitarians

Features

Quick Mapping

Linked to the common operational datasets to automatically generate maps. Just include PCodes.

Intelligent Suggestions

HXLDash uses HXL to give intelligent suggestions which allows dashboards to be built in 2 minutes.

Print Friendly

50% of data products are printed hard copies in the field. HXLDash provides easy print formats for all dashboards.

Low Bandwidth

HXLDash aims to make dashboards 10% the filesize as other popular providers so they work in low bandwidth areas

Free and Open Source

HXLDash is free to use with no sign up required and the code is open source for everybody to access.

Integrates with the HXL ecosystem

HXLDash integrates seamlessly with the HXL ecosystem such as the HXL Proxy to create fast automated workflows

Watch HXLDash in action

Create an online sharable dashboard in less than 2 minutes. Watch the example video below to see how easy it is.

Data Security

For the Alpha test of this product the team has decided to host no data on the HXLDash server. To find out more about this approach, it's limitations and plans for the future read more here.
